# Fibula Travel Agency

Fibula Travel Agency is a simple and responsive front-end project built with HTML and CSS.  
It showcases travel packages, featured destinations, and a clean landing page layout for a travel agency website.

## Features
- Clean landing page design
- Travel packages section
- Featured itineraries
- Destination images with captions
- Simple and organized structure

## Technologies Used
- HTML5
- CSS3
